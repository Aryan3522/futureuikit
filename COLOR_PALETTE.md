# Future UI - Premium Color Palette

This palette is designed for a modern, high-end "Pro" aesthetic. It emphasizes clarity, depth, and professional motion.

## Core Colors

| Role | Color | Hex | Tailwind Equivalent | Description |
| :--- | :--- | :--- | :--- | :--- |
| **Primary** | Indigo | `#6366f1` | `indigo-500` | Main action color, buttons, active links, and brand highlights. |
| **Secondary** | Cyan | `#0ea5e9` | `sky-500` | Accent color used for secondary highlights and subtle gradients. |
| **Dark Base** | Slate | `#09090b` | `zinc-950` | The deep, high-contrast base for dark mode layouts. |
| **Neutral** | Zinc | `#71717a` | `zinc-400` | Muted text, secondary labels, and soft borders. |
| **Text** | Pure | `#fafafa` | `zinc-50` | Primary foreground text for maximum readability. |

## Application Rules

1.  **Sparingly:** Use the Primary Indigo only for elements that require attention (CTAs, Active Tabs).
2.  **Depth:** Use the Secondary Cyan as a companion to Indigo in gradients to create a sense of light and dimension.
3.  **Contrast:** Maintain high contrast between the Slate background and Zinc text for an accessible, premium experience.
4.  **Borders:** Use Zinc-800/900 for borders to keep them felt but not seen.
